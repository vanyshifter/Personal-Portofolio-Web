# Personal Portfolio Website

## Description
This project is a simple personal portfolio website developed as an individual academic project.
The website introduces personal background and showcases basic portfolio content using visual elements.
The focus of this project is on basic web layout, navigation structure, and content presentation.

## Features
- Personal introduction section
- Portfolio image display
- Basic navigation menu
- Static web layout

## Technologies Used
- HTML
- CSS
- Visual Studio Code

## Project Type
Individual Academic Web Project
